Installation Guide


Q&A

Why won't Windows let me run the game?
Windows blocks the game because it is not signed. To bypass this, just click on more info, then click run anyway.
Where is the game?
It's in the folder that you downloaded. Double click on "HirokiPoops.exe."
How to run on Linux?
These instructions will work on most Linux distros. Go to the folder where you extracted the zip archive. Right click the file "HirokiPoops," and click properties. Go to the permissions tab and hit "Allow executing as program."
Why aren't you giving out the direct link?
I want this game to stay somewhat private.
