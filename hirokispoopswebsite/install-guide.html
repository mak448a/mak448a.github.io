Hiroki's Poops Installation Guide

This guide will walk you through the installation of Hiroki's Poops. Click if you're on Windows, and click this link if you're on Linux.

Windows Installation

  1. Click this link, then save the file to your downloads folder.
  2. Open the downloads folder in the files app.
  3. Right-click the file and click "Extract...."
  4. In the popup window, click "Extract Here."
  5. Go to the extracted folder, then go into the folder inside.
  6. Double click the HirokiPoops.exe file.
  7. If Windows blocks the app, click more info, then click run anyway.
  8. You're done! :)

Linux Installation

If you're using Linux, I'm assuming that either you're using Nautilus, or know how to extract a file with your file manager. Ok, let's get started.

  1. Click this link, then save the file to your downloads folder.
  2. Open the downloads folder in the files app.
  3. Right-click the file and click "Extract Here."
  4. Go into the extracted folder.
  5. Right click an empty space in the file window, and click "Open in Terminal."
  6. Once the Terminal is open, execute this command: $ chmod +x HirokiPoops
  7. Close the Terminal window. Go back to your files app and double click on the "HirokiPoops" file.
  8. You're done! :)
